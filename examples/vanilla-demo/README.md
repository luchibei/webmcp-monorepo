# Vanilla WebMCP Demo

Pure HTML demo with two tools:

- `getPageTitle`
- `searchFlights` (mock data)

## Run

From repo root:

```bash
npx serve examples/vanilla-demo
```

Open the printed URL (usually `http://localhost:3000`).

Alternative (Python):

```bash
python -m http.server 5173 --directory examples/vanilla-demo
```

Then open `http://localhost:5173`.
